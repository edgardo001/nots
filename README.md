# Notas Post-It

Aplicación de notas estilo Post-It con React 19 + TypeScript + Bun.

## Stack

- **Frontend**: React 19 + TypeScript + Vite 6
- **Runtime**: Bun
- **Estado**: Zustand
- **Storage**: IndexedDB (idb)
- **Drag & Drop**: @dnd-kit

## Desarrollo

```bash
cd src/frontend
bun run dev
```

## Build

```bash
cd src/frontend
bun run build
```

## Estructura

```
src/
├── frontend/     # React app
└── docker/       # Docker config
```
